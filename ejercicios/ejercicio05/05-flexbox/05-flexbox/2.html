📌 ENUNCIADO EJERCICIO 2:
- Aplica Flexbox para lograr la siguiente estructura:
1. La cabecera (.header) debe ser un contenedor flex horizontal que alinee el logo a la izquierda y los enlaces a la derecha.
2. El contenedor .main-layout debe ser un flex container en fila (dos columnas).
3. La barra lateral (.sidebar) debe tener un ancho fijo de 240px (puedes usar flex-basis o width).
4. El área de perfil (.profile-main) debe ocupar todo el espacio restante (flex-grow).
5. La cabecera y el contenido deben apilarse verticalmente (esto ya es natural, pero asegúrate de que el contenedor principal .profile-app no tenga flex en columna a menos que lo necesites; el reto es el layout de dos columnas y la cabecera).
